# VuePokemons

Nesta pequena aplicação com VueJs3 e Bootatrap 5 você poderá ver minhas habilidades com VueJs3, componentização, clean code, consumo de API e testes unitários.

## Tecnologias
* Vue
* Vue Router
* Vite
* Vitest
* Bootstrap 5
* Pokeapi
* Chartjs

## Instalação
```
    npm install
```

## Execução
```
    npm run dev
```

## Testes
```
    npm run test
```
